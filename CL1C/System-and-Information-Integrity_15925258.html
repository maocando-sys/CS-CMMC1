CMMC LVL 1 CS : System and Information Integrity

Created by Manuel Ocando on Feb 03, 2026

1. Purpose

This policy ensures that systems handling Federal Contract Information (FCI) are protected from malicious code, vulnerabilities, and integrity threats, in accordance with CMMC Level 1.

2. Scope

Applies to:

  • All systems handling FCI

  • All organizational users

3. Policy Statements

3.1 Flaw Remediation

Identified system flaws are remediated in a timely manner through patching or configuration updates.

3.2 Malicious Code Protection

Systems are protected using approved malicious code protection mechanisms.

3.3 Monitoring

Systems are monitored to detect potential integrity or security events.

4. Roles and Responsibilities

  • System Administrators maintain protections and updates

  • Users report suspicious activity

5. Enforcement

Failure to comply with integrity requirements may result in access restrictions or disciplinary action.

6. Compliance Mapping

  • SI.L1-3.14.1 – Identify and remediate flaws

  • SI.L1-3.14.2 – Malicious code protection

  • SI.L1-3.14.4 – Update protection mechanisms

  • SI.L1-3.14.5 – System monitoring

7. Review

Reviewed annually or upon major incidents or system changes.
